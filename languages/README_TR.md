# MdgWa

## Özellikler

### Özelleştirme

|  | Durum |
| ------------- | ------------- |
| Yeni WhatsApp teması | ✅ |
| Yeni ayarlar stili | ✅ |
| Sohbetleri filtrele | ✅ |
| Düğmelerin kenarlığı | ✅ |
| Menü simgeleri | ✅ |
| Çizgili simgeler | ✅ |
| İsim ve biyografi göster | ✅ |
| ÖÖ:DD Zaman Formatı | ✅ |
| Zaman içinde saniyeleri göster | ✅ |
| Global renk değiştirme | ✅ |
| Sohbetleri ve grupları ayır | ✅ |
